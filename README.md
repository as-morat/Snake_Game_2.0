# Snake Game 🐍

A classic Snake game implementation with score tracking, pause functionality, and multiple difficulty levels.

![image](https://github.com/user-attachments/assets/acbaa9a9-5ba2-4cf1-a031-915a41dc7cf3)


## Features

- 🎮 Classic Snake gameplay
- 📊 Score and length tracking
- ⏸️ Pause/resume functionality
- 🔄 Restart option after game over
- 🎚️ Adjustable difficulty levels (Easy by default)

## How to Play

### Controls
- **Arrow Keys**: Move the snake (Up/Down/Left/Right)
- **Enter**: Pause/Resume the game
- **Space**: Restart after game over

### Game Rules
- Eat the food (🍎) to grow longer and increase your score
- Avoid hitting your own tail
- The game ends when the snake collides with itself or the wall
